####轻量级学习型框架
----
#####主要是整合自己的思路，来构建如何完成一个简单的MVC。通过实践能从中去理解这样的一种设计思路，再去想想，其实很多小型框架，无非也就这样的一种加载理念，和设计思路。只是在一种简单的设计思路上，扩展整个活动空间，完善一个框架系统。
#####1、自动加载函数库
#####2、自动注册实例化类
#####3、单一入口，只加载核心文件，就可以启动
#####4、三层结构，controller,Model,Service
#####5、smarty缓存机制
